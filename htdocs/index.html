rquota

quota and repquota for NFS and Lustre
rquota includes quota and repquota utilities that support NFS and Lustre file system quotas.

rquota is a drop-in replacement for the Red Hat quota RPM, if you don't need quota support for local file systems.

rquota only queries file systems listed in /etc/quota.conf.

rquota reports usage and quota limits in human readable quantities.

rquota file system names can be aliased for readability in /etc/quota.conf.

See SourceForge summary page for downloads, etc..
